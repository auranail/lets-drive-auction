# Let's Drive Auction

A professional, modern website for premium car auctions. Built with clean, minimalist design inspired by Apple's design philosophy.

## Features

- **Clean Navigation** - Sticky header with smooth scrolling
- **Hero Section** - Compelling call-to-action with featured buttons
- **Featured Auctions** - Display active auction listings with real-time bidding
- **How It Works** - 4-step process explanation for auction participation
- **Why Choose Us** - 6 key features highlighting platform benefits
- **Newsletter Signup** - Email subscription for auction alerts
- **Contact Section** - Contact information and inquiry form
- **Professional Footer** - Links, social media, and legal information

## Design

- **Color Scheme**: Black and white minimalist design
- **Typography**: System font stack for optimal performance
- **Responsive**: Fully responsive design for mobile, tablet, and desktop
- **Animations**: Smooth transitions and hover effects

## Files

- `index.html` - Main website structure
- `styles.css` - Complete styling and responsive design

## Getting Started

1. Open `index.html` in your web browser
2. Navigate through sections using the top menu
3. Explore featured auctions and company features

## Responsive Design

The website is fully responsive:
- **Desktop**: Full navigation and multi-column layouts
- **Tablet**: Optimized touch interactions
- **Mobile**: Single-column layout with streamlined navigation

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
